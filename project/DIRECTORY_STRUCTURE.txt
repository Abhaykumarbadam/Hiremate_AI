Key Directories Explained
==========================

📁 src/pages/
   Contains all page components for the 4-page flow:
   1. HomePage - Landing with "Start Interview" button
   2. ResumeAnalyzer - Upload and analyze resume
   3. InterviewScreen - AI interview with Q&A
   4. ResultsPage - Scores and evaluation

📁 src/config/
   API endpoint configuration
   - Centralized API URLs
   - Easy to update for production

📁 src/types/
   TypeScript type definitions
   - API response interfaces
   - Component prop types
   - Type safety throughout app

📁 src/utils/
   Utility functions
   - PDF generation logic
   - Reusable helper functions

Quick Start
===========

1. Place backend.py in project root
2. npm install
3. pip install [dependencies from BACKEND_SETUP.md]
4. export GEMINI_API_KEY="your-key"
5. ./start.sh (or start.bat on Windows)
6. Open http://localhost:5173
